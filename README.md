# Draw.io diagrams to SVG

## Requirements

- Node.js
- Draw.io
- Inkscape

## Guide

1. Create two folders for input and output
```sh
mkdir _input_ _output_
```

2. Place `.xml` or `.drawio` files inside of `_input_`

3. Install the dependencies

```sh
npm i
```

4. Build the application

```sh
npm run build
```

5. Run it

```sh
npm start
```
